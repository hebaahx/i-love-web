# i-love-web

Learning Journal en Digital Gardening

De instructie voor deze leertaak vind je in: [INSTRUCTIONS](https://github.com/fdnd-task/i-love-web/blob/main/docs/INSTRUCTIONS.md)

### Welkom bij mijn Learning Journal!
In deze repository verzamel ik belangrijke informatie en bronnen uit de lessen. Ook gebruik ik deze plek om te oefenen met I Love Web-opdrachten en te experimenteren met HTML, CSS en JavaScript.

### Wat vind je hier?

- Mijn aantekeningen en inzichten

- Bronnen en links uit de lessen

- Kleine code-experimenten en oefenopdrachten

- Mijn groei en ontwikkeling als webdeveloper

### Doel:

Deze repo helpt mij om mijn leerproces bij te houden, structuur te behouden en steeds beter te worden in coderen.
